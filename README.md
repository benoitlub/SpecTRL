# SpecTRL V.0.2.1 — by Marty

Experimental Trace Resonance Logger.

SpecTRL is a playful paranormal-style scanner built from the Creature-Sync base, but kept as a separate project. It uses real mobile-friendly signals — microphone, camera, motion sensors and future environmental sensors — then turns them into narrative interpretations: traces, remanences, unstable signatures and suspicious Marty-grade conclusions.

## Core principle

Real sensors. Fictional interpretation. No fake scientific claim.

SpecTRL should never present a result as proof of ghosts. It should speak in terms of:

- acoustic signatures,
- visual structures,
- remanence,
- environmental traces,
- unstable hypotheses,
- Marty confidence.

## V.0.2.1 scope

This version keeps the Creature-Sync technical base intact while shifting the identity toward SpecTRL.

Immediate goals:

- rename the app identity from Creature-Sync to SpecTRL,
- preserve the current audio scanner,
- prepare the future sensor layer,
- prepare the Marty SLS Layer concept,
- avoid breaking the existing build.

## Planned sensors

- Microphone: acoustic trace analysis.
- Camera: visual scan and future Marty SLS Layer.
- Accelerometer / gyroscope: motion and stability.
- Magnetometer: spectral compass / field disturbance.
- Light sensor: shadow and luminosity changes.
- GPS: scan journal context.

## Marty SLS Layer

Future camera mode inspired by SLS-style stick figures.

The app may display skeletal overlays for human-like visual structures. It should frame them as unstable visual signatures, not confirmed entities.

Example wording:

> Humanoid structure detected.  
> Stability: low.  
> Scientific confidence: 8%.  
> Marty confidence: 93%.
